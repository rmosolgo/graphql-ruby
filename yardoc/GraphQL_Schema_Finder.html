Class: GraphQL::Schema::Finder < Object Find schema members using string paths Examples:

Finding object types

MySchema.find(“SomeObjectType”)

Finding fields

MySchema.find(“SomeObjectType.myField”)

Finding arguments

MySchema.find(“SomeObjectType.myField.anArgument”)

Finding directives

MySchema.find(“@include”) Instance methods: find, find_in_directive, find_in_enum_type, find_in_field, find_in_fields_type, find_in_input_object, find_in_type, initialize
