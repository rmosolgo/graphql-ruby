Module: GraphQL::Tracing::StatsdTrace A tracer for reporting GraphQL-Ruby times to Statsd. Passing any Statsd client that implements .time(name) { ... } will work. Examples:

Installing this tracer

eg:

$statsd = Statsd.new ‘localhost’, 9125

class MySchema < GraphQL::Schema use GraphQL::Tracing::StatsdTrace, statsd: $statsd end Includes: GraphQL::Tracing::PlatformTrace Instance methods: initialize, platform_authorized, platform_authorized_key, platform_execute_field, platform_field_key, platform_resolve_type_key
