Class: GraphQL::StaticValidation::ValidationContext < Object The validation context gets passed to each validator. It exposes a GraphQL::Language::Visitor where validators may add hooks. (Language::Visitor#visit is called in Validator#validate) It provides access to the schema & fragments which validators may read from. It holds a list of errors which each validator may add to. It also provides limited access to the TypeStack instance, which tracks state as you climb in and out of different fields. Extended by: Forwardable Instance methods: initialize, on_dependency_resolve, validate_literal
