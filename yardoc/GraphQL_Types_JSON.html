Class: GraphQL::Types::JSON < GraphQL::Schema::Scalar An untyped JSON scalar that maps to Ruby hashes, arrays, strings, integers, floats, booleans and nils. This should be used judiciously because it subverts the GraphQL type system. Use it for fields or arguments as follows: field :template_parameters, GraphQL::Types::JSON, null: false argument :template_parameters, GraphQL::Types::JSON, null: false Class methods: coerce_input, coerce_result
