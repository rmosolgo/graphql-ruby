Class: GraphQL::Query::Partial < Object This class is like a GraphQL::Query, except it can run on an arbitrary path within a query string. It depends on a “parent” Query. During execution, it calls query-related tracing hooks but passes itself as query:. The Partial will use your Schema.resolve_type hook to find the right GraphQL type to use for object in some cases.
