Class: GraphQL::Schema::Timeout::TimeoutError < GraphQL::ExecutionError This error is raised when a query exceeds max_seconds. Since it’s a child of GraphQL::ExecutionError, its message will be added to the response’s errors key. To raise an error that will stop query resolution, use a custom block to take this error and raise a new one which doesn’t descend from GraphQL::ExecutionError, such as RuntimeError.
