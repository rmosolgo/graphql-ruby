Overview

This module is part of a private API. You should avoid using this module if possible, as it may be removed or be changed in the future.

This object can be passed like an array, but it doesn’t allocate an array until it’s used.

There’s one crucial difference: you have to capture the result of #<<. (This works with arrays but isn’t required, since it has a side-effect.)
